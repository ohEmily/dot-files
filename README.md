### new machine:
1. setup `~/.env` based on [`.env.example`](https://github.com/ohEmily/dot-files/blob/main/.env.example)
2. one line install:
```
curl -fsSL https://raw.githubusercontent.com/ohEmily/dot-files/main/install.sh | bash
```

### linting:
```
brew install shellcheck
shellcheck install.sh
```
